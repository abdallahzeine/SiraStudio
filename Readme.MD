## ✨ Overview

SiraStudio is a CV builder web app for creating, editing, and exporting professional resumes with reusable layouts and presets.

## 🚀 Features

- 🎨 Multiple CV templates
- 👀 Real-time preview
- 🧩 Customizable sections
- 🗂️ Layout presets
- 💾 Local data storage
- 🖨️ Print-friendly output
- 📱 Responsive UI
- 🖱️ Drag and drop reordering

## 🛠️ Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:5173` after the dev server starts.

## 📦 Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
